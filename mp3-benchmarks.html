MP3 File Benchmarks

Use of compression with the -z 1, -z 7, and -z 9 gpg options did not lead to a significant change in file size for mp3 files. crypto disables compression for mp3 files (and all other binary files) by default in order to improve encryption performance.

The mp3 file tests were performed on a directory that contained ten mp3 files that ranged in size from 3.5MB to 4.3MB.

The total CPU time for the GPG Default, crypto Default, and crypto --speed runs were 2.14, 0.80, and 0.77 sec, respectively.
